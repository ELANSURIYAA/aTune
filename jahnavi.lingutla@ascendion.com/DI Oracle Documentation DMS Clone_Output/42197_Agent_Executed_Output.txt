## Procedure Overview

**Procedure Name:** `bronze.usp_Load_bronze_Layer_Full`

**Schema:** bronze

**Purpose:** Orchestrates the complete ETL pipeline for loading all bronze layer tables with full refresh. This master procedure coordinates the execution of 12 individual table load procedures and provides comprehensive logging and audit trail functionality.

- **Type:** Stored Procedure
- **Parameters:** None (Parameterless procedure)
- **Return Type:** None
- **Transaction Control:** Includes COMMIT statements
- **Error Handling:** Comprehensive exception handling with audit logging

### 3. Summary Statistics Calculation
After all table loads complete, the procedure queries the audit log table to calculate:

**Success Metrics:**
```sql
SELECT COUNT(*), NVL(SUM(records_inserted), 0)
FROM bronze.bz_Audit_Log
WHERE batch_id = v_BatchID AND status = 'SUCCESS'
```

**Failure Metrics:**
```sql
SELECT COUNT(*), NVL(SUM(records_failed), 0)
FROM bronze.bz_Audit_Log
WHERE batch_id = v_BatchID AND status = 'FAILED'
```

### 4. Execution Time Calculation
Calculates total execution time in seconds using timestamp arithmetic:
```sql
v_ExecutionTime := EXTRACT(DAY FROM (v_EndTime - v_StartTime)) * 86400 + 
                   EXTRACT(HOUR FROM (v_EndTime - v_StartTime)) * 3600 + 
                   EXTRACT(MINUTE FROM (v_EndTime - v_StartTime)) * 60 + 
                   EXTRACT(SECOND FROM (v_EndTime - v_StartTime))
```

### 5. Completion Logging
Outputs comprehensive execution summary including:
- End timestamp
- Total execution time
- Tables processed/succeeded/failed counts
- Row-level statistics (inserted/failed)

### 6. Master Audit Record Creation
Inserts a master audit record into bronze.bz_Audit_Log with:
- Source: 'source_layer.*'
- Target: 'bronze.*'
- Load type: 'FULL_REFRESH_ALL_TABLES'
- All execution metrics and statistics
- Batch ID for traceability

### 7. Transaction Commit
Commits all changes to persist audit records.

## Error Handling

### Exception Block
When any error occurs during execution:

1. **Error Capture:**
   - Error message (SQLERRM)
   - Error code (SQLCODE)
   - Sets severity, state, and line (initialized to 0)

2. **Execution Time Calculation:**
   - Calculates time up to error occurrence

3. **Status Update:**
   - Sets v_OverallStatus to 'FAILED'

4. **Error Logging:**
   - Outputs detailed error information to DBMS_OUTPUT
   - Displays error number, severity, state, line, and message

5. **Error Audit Record:**
   - Inserts error record into bronze.bz_Audit_Log
   - Includes error details in error_message field
   - Format: 'Error [code]: [message]'

6. **Transaction Commit:**
   - Commits error audit record

7. **Error Propagation:**
   - Re-throws error using RAISE_APPLICATION_ERROR(-20001, message)

3. **Migration Order:**

   - Migrate this master procedure last

4. **Testing Post-Migration:**
   - Verify all child procedures are valid

5. **Compatibility Notes:**
   - Uses Oracle-specific features (SYSTIMESTAMP, EXTRACT, DBMS_OUTPUT)
