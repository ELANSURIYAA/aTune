Reviewer Validation Checklist

Completeness Checks

- Verify that all mandatory sections specified in the agent instructions are present in the output.
- Confirm that all required fields and blocks from the expected JSON output template are included.
- Ensure that all configuration, mappingResults, entityScores, and aiInsights sections are generated, even if input files are missing.
- Check that all required headers, classifications, and tags are present as per the agent instructions.
- Validate that the output includes handling for missing input files, as per the agent’s error-handling requirements.
- Confirm that no referenced section from the agent instructions is omitted in the output.
- Ensure that all formatting and structural requirements from the instructions are followed.

Accuracy Checks

- Confirm that the output aligns strictly with the agent instructions and expected output template.
- Verify that no hallucinated or fabricated content is present in the output.
- Ensure that error handling for missing input files is correctly implemented and justified in the output.
- Check that all terminology and field names are used consistently and as specified in the instructions.
- Validate that no unsupported assumptions or content are introduced due to missing input files.
- Confirm that the output does not contradict any requirements or constraints from the agent instructions.
- Ensure that the output structure and field values match the expected JSON schema exactly.

Efficiency Checks

- Verify that the output does not include redundant or duplicate sections.
- Ensure that the output remains concise and does not contain unnecessary verbosity or filler content.
- Confirm that no unnecessary or repeated error messages are generated for missing input files.
- Check that the output structure is optimized for readability and review efficiency.
- Ensure that only required error handling and minimal necessary content are included when input files are missing.
- Validate that the output is token-efficient and avoids repeating information from the agent instructions.
