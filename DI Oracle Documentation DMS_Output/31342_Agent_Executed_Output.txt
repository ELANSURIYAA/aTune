Overview of Program\n\nThe bronze.usp_Load_bronze_Layer_Full stored procedure orchestrates a comprehensive ETL pipeline that performs full refresh data loads across twelve bronze layer tables. The procedure sequentially invokes individual table-specific load procedures, passing a batch identifier for tracking and audit purposes. It captures execution metrics including row counts, processing times, and success/failure statistics, logging all activities to the bronze.bz_Audit_Log table.
